# FrizBank React

Aplicação React baseada no FrizBank original, migrada de JavaScript vanilla para React com Vite.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Estrutura do Projeto

- `src/components/` - Componentes React
- `src/hooks/` - Custom hooks
- `src/utils/` - Funções utilitárias
- `src/assets/` - Assets estáticos
- `public/` - Arquivos públicos

## Funcionalidades

- Landing page
- Login/Registro
- Reconhecimento facial (Face API)
- Dashboard com gráficos
- Gerenciamento de criptomoedas
- Modo escuro
